## AI에게 넘기는 방법

위 질문에 답변을 다 작성했으면, 아래처럼 AI에게 프롬프트를 작성하세요:

```
아래는 우리 EcoBid 프로젝트의 제품 기획 답변이야.
이 내용을 기반으로 [원하는 작업]을 해줘.

[답변 전체 복사 붙여넣기]
```

### 프롬프트 예시들

**백엔드 세팅:**
```
아래는 우리 EcoBid 프로젝트의 기획 답변이야.
이 내용을 기반으로 Express + Sequelize 백엔드를 세팅해줘.
User 모델이랑 회원가입/로그인 API를 JWT 방식으로 만들어줘.

1. 회원가입: 이메일, 비밀번호, 닉네임, 학과
2. 비밀번호 조건: 최소 8자
3. 로그인: 이메일+비밀번호, 토큰 7일 유지
...
```

**React Navigation 도입:**
```
현재 우리 App.tsx에서 useState로 화면 전환하고 있는데,
React Navigation (Stack + Bottom Tabs)으로 바꿔줘.
탭은 홈, 지도, 미션, 채팅, 마이페이지 5개야.
기존 screens/ 폴더의 컴포넌트들을 그대로 연결해줘.
```

**API 연동:**
```
현재 HomeScreen.tsx에서 상품 목록이 하드코딩이야.
GET /api/products API를 호출해서 실제 데이터를 보여주도록 바꿔줘.
axios를 사용하고, 로딩 상태와 에러 처리도 추가해줘.
```
